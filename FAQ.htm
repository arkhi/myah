À propos de ce site

Qu'est ce que le myah ?
Il s'agit d'un projet visant à recomposer spatialement l'adresse de chacune des machines se connectant à lui.
Chaque ordinateur sur Internet ayant une adresse IP (statique ou dynamique), il s'agit de situer cette machine parmi d'autres. Dans le cas présent, cela revient à attribuer un pseudonyme à une localisation dans un espace virtuel.
À quoi correspondent les tailles et positions de chaque élément ?
Elles résultent en partie de l'adresse IP. Disons que chaque entité se nourrit du visiteur qu'elle abrite. Par analogie, une maison laissée à l'abandon devient une ruine là où une maison habitée s'enrichit (d'une façon ou d'une autre) des attributs que voudront bien lui donner ses occupants.
Quel est le but du myah ?
Pour l'instant aucun. Il ne s'agit là que d'un monde virtuel en phase d'expérimentation.

À propos du VRML

Qu'est ce que le VRML ?
VRML, traduit en français, ça donne à peu près ça : Langage de Modélisation de la Réalité Virtuelle. C'est un langage pour créer des objets en trois dimensions, et les exploiter sur un navigateur Internet.
On parle de monde VRML pour désigner une page Web utilisant ce langage.
Comment faire pour accéder à un monde VRML ?
Le VRML n'est pas intégré de façon native dans les navigateurs Internet. Pour afficher une page VRML, il faut installer un plugin (ou extension), qui vous permettra d'interagir avec les objets en trois dimensions de la page.
Une fois le plugin installé, il faut généralement redémarrer le navigateur Internet.
Après cela, il suffit de visionner un monde VRML.
Où télécharger un plugin VRML ?
Vous en trouverez sur la page liens.
Certains plugins vous proposent de s'installer automatiquement sur votre navigateur. Il est parfois possible de télécharger un fichier exécutable, et de les installer plutôt par ce biais là.
Je n'arrive pas à installer ce plugin, pourquoi ?
Parce que si tout marchait correctement, il n'y aurait pas besoin de cette FAQ
Les plugins VRML ne sont pas tous parfaits, et peuvent dans certains cas ne pas fonctionner correctement. La première solution est de vous adresser à l'éditeur de ce plugin et de vous armer de patience. La seconde solution est d'installer un autre plugin qui, avec un peu de chance, marchera mieux.
En dernier recours, préférez l'installation d'un plugin depuis un fichier exécutable plutôt que directement depuis le navigateur.

Comment utiliser les barres d'outils ?

Avec la souris, ça serait un bon début. Mais voyons plus en détail les possibilités :

Les modes de déplacement
Dans ce mode de déplacement, vous êtes lié au plan du sol, vous évoluez en deux dimensions dans un environnement en trois dimmension (c'est dommage, hein ?) ; les objets 3D sont considérés comme solide et vous ne pouvez les traverser ; si on vous mettait un fusil entre les mains, vous vous croiriez dans un FPS.
Dans ce mode de déplacement, vous pouvez vous déplacer dans n'importe quelle direction, et vous élever dans les airs pour découvrir votre environnement en trois dimensions d'une façon tout à fait nouvelle. Par contre, les objets 3D n'ont plus de consistance dans ce mode, et vous pouvez passer à travers sans le moindre problème.
Dans ce mode de déplacement, ce n'est plus vous qui vous déplacez, mais c'est l'environnement qui tournera devant vous.
Avatar : Disponible uniquement via le menu contextuel, cette fonction affiche une représentation de votre présence dans la scène (un avatar), si elle est disponible. Vous pouvez faire l'analogie à un jeu vidéo à la troisième personne.
Les options de déplacement
Cette option de déplacement permet de se déplacer sur un plan horizontal, vers l'avant et l'arrière, ou de pivoter sur la gauche ou la droite. En mode Walk, le plan horizontal sur lequel vous vous déplacerez sera le sol. En mode Fly, le plan horizontal sur lequel vous vous déplacerez sera ré-orienté en fonction de votre angle de vue.
Cette option vous permet de vous déplacer sur un plan dont l'orientation dépend de votre mode de déplacement : en mode Walk, vous vous déplacerez sur un plan horizontal, comme l'option Plan, sauf que vous ne pivoterez pas sur les cotés, mais vous vous déplacerez latéralement. En mode Fly, vous vous déplacerez sur un plan vertical ; vous pourrez alors vous déplacer vers le haut ou le bas, ou latéralement, sans modifier votre angle de vue.
Cette option vous permet de tourner votre point de vue, soit horizontalement, soit verticalement. En mode Fly, cette option vous permettra de modifier le plan horizontal sur lequel vous évoluez avec l'option Plan.
Cette option est désactivée en mode Walk. En mode Fly, elle permet d'effectuer une rotation de l'angle de vue (le roulis), dans le sens des aiguilles d'une montre ou dans le sens contraire.
Les points de vue
Cette fonction vous permet de vous déplacer d'un point de vue prédéfini à un autre. Vous pouvez utiliser les flèches pour passer au point de vue suivant ou précédent, ou cliquer sur view pour afficher la liste des points de vue disponibles, puis sélectionner celui auquel vous souhaitez vous rendre.

Comment utiliser le menu contextuel ?

En faisant un clic-droit sur la fenêtre d'environnement. VRML, pardi !
Voici le détail des options :

Viewpoints
Ouvre un menu déroulant qui permet de choisir un nouveau point de vue. Voir les points de vue.
Headlight
Permet d'activer ou de désactiver l'éclairage frontal. Si l'option est désactivée, l'environnement risque de vous paraître bien sombre.
Navigation
Permet de choisir le mode de déplacement.
Speed
Permet de modifier la vitesse de déplacement. Cinq choix sont possibles, en allant du plus lent (Slowest) au plus rapide (Fastest)
Full Screen
Permet de modifier l'affichage de la fenêtre VRML en forçant la résolution à une valeur définie. Attention, si vous choisissez les valeurs de votre résolution d'ecran actuelle, les barres d'outils et le cadre de la fenêtre disparaitront. Pour revenir à la résolution d'origine, il suffit de repasser par le mode contextuel et de choisir Full Screen : Windowed view, ou d'appuyer sur la touche Echap
Show / Hide Toolbars
Permet d'afficher / cacher les barres d'outils.
Show / Hide Console
Permet d'afficher / cacher la console du plugin (pour utilisateurs avancés).
Preferences...
Permet de choisir les préférences d'utilisation du plugin.
Help
Permet d'afficher les informations de la version du plugin, ou d'accéder au manuel de l'utilisateur (en anglais).
Quand je charge la page, tous les blocs apparaissent en blanc ; c'est normal ?
Ça dépend ce que vous appelez normal. Certains ordinateurs ont des difficultés avec l'accélération 3D par défaut (DirectX). Dans ce cas, il convient d'aller dans les préférences du plugin, et de choisir un autre moteur de rendu (OpenGL, par exemple).

J'ai trop joué avec les commandes, et maintenant je ne vois plus que du blanc (ou une autre couleur). J'ai cassé quelque chose ?
Vous n'avez rien cassé, mais vous êtes très probablement coincé face à un mur ; pour les voltigeurs, vous êtes surement face au sol...
Plusieurs solutions pour vous sortir de là. Soit vous changez de point de vue. Soit vous utilisez la fonction Restore qui vous ramènera à votre dernier point de vue. Les plus vadrouilleurs apprécieront de jouer avec les commandes Align, Fit et Goto pour revenir à leur point de départ par leurs propres moyens.
J'ai changé de point de vue, mais dès que je me déplace, je me retrouve avec un écran blanc/rose/vert/bleu (rayez la mention inutile). Que se passe t'il ?
Certains points de vue vous emmènent en hauteur. Le problème, si vous êtes en mode Walk, c'est qu'au moindre mouvement, vous serez ramené au sol, avec une vue plongeante sur ce dernier. Reportez vous au paragraphe des gens perdus pour voir les solutions qui vous permettront de vous réorienter.
Je suis bloqué ! Je n'arrive plus à avancer au delà d'une limite invisible. Comment sortir de là ?
Vous, vous avez trop joué à passer du mode Fly au mode Walk... Résultat, en mode Fly, vous êtes passé à travers un bloc. À l'intérieur de ce bloc, vous êtes repassé en mode Walk, en oubliant deux choses : En mode Walk, vous ne pouvez plus passer à travers les blocs ; et accessoirement, quand vous êtes dans un bloc, les parois de ce bloc ne vous sont plus visibles. Repassez en mode Fly pour traverser le mur, ou changez de point de vue (voir également la fonction Restore).

Navigation au clavier :

À force de naviguer avec la souris, j'ai une crampe au doigt. Je peux avoir un arrêt de travail avec ça ?

C'est peu probable, vu qu'il est également possible de naviguer au clavier, avec les touches du clavier. Voici déjà celles qui sont le plus pratiques :

Haut, bas, gauche, droite
Pas de secret, ces touches servent à avancer, reculer, aller à gauche ou à droite, voir monter ou descendre suivant les touches que l'on utilise en même temps.
Page suivante, page précédente
Sert à passer d'un point de vue au suivant (ou au précédent).
Espace
Permet de passer en option Turn tant que l'on appuie dessus. Se réorienter avec le pavé directionnel.
Alt
Permet de passer en option Pan tant que l'on appuie dessus. Se réorienter avec le pavé directionnel.
Alt+Espace (Mode Fly uniquement)
Permet de passer en option Roll tant que l'on appuie dessus. Se réorienter avec le pavé directionnel.
Majuscule
Permet d'accélérer le déplacement de manière assez significative.
Crtl
Permet d'accélérer encore plus le déplacement
Combiner avec "Majuscule" pour augmenter la vitesse.
1 (Mode Fly uniquement)
Permet de faire un Roll dans le sens des aiguilles d'une montre.
2 (Mode Fly uniquement)
Permet de faire un Roll dans le sens inverse des aiguilles d'une montre.
3
Permet de faire un Turn vers le bas.
4
Permet de faire un Turn vers le haut.
F
raccourci pour la fonction Fit.
